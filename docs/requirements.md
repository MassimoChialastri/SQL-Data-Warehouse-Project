## Project Requirements 📋

### Building the Data Warehouse 

#### Objective
Design and implement a modern Data Warehouse on SQL Server to centralize sales information and support analytical reporting and strategic decision-making.

#### Specifications
- **Data Sources**: Load data from Olist source system delivered in CSV format.
- **Data Quality**: Perform data cleansing and address inconsistencies before carrying out any analysis.
- **Data Optimization**: Refine and optimize the source data into an efficient model suitable for analytical workloads.
- **Scope**: Track historical data to allow business users to analyze trends and data evolution over time.
- **Documentation**: Produce comprehensive documentation of the data model to assist business users and analytics teams.

---

### BI: Analytics & Reporting (Data Analysis)

#### Objective
Develop Power BI analytics to provide in-depth insights into:
- **Customer Behavior**
- **Product Performance**
- **Sales Trends**

These insights equip stakeholders with essential business metrics, supporting data-driven strategic decisions.
